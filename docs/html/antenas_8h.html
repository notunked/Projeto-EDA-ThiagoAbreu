char  frequencia,

)

Insere uma nova antena na lista ligada de forma ordenada.

A antena é inserida mantendo a ordem crescente das coordenadas (x, y). Se já existir uma antena na mesma posição, a inserção é ignorada.

Parâmetros
listaApontador para o início da lista ligada de antenas.
frequenciaLetra que identifica a frequência da nova antena.
xCoordenada X da nova antena.
yCoordenada Y da nova antena.
Retorna
Apontador para o novo início da lista ou NULL em caso de erro ou duplicada.

Insere uma nova antena na lista ligada de forma ordenada.

A função verifica se já existe uma antena na posição especificada. Caso contrário, aloca memória para um novo nó e insere-o mantendo a ordem da lista baseada nas coordenadas.

Parâmetros
listaApontador para o início da lista ligada de antenas.
frequenciaCaractere que representa a frequência da antena.
xPosição X da antena.
yPosição Y da antena.
Retorna
Novo apontador para o novo início da lista, ou NULL em caso de erro ou duplicada.
