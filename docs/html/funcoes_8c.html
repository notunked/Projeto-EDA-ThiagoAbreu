Este módulo fornece funcionalidades para:

  • Carregar uma lista de antenas a partir de ficheiro
  • Remover antenas da lista ligada
  • Verificar e adicionar posições
  • Libertar memória de listas de antenas ou coordenadas

Carrega antenas a partir de um ficheiro de texto.

Carrega uma lista de antenas a partir de um ficheiro de texto.

Lê caractere a caractere e converte o conteúdo em uma lista ligada de antenas. Cada antena é representada por um caractere diferente de '.' na matriz textual.

Parâmetros
nomeFicheiroCaminho para o ficheiro de entrada.
linhasApontador para armazenar o número de linhas.
colunasApontador para armazenar o número de colunas.
Retorna
Apontador para o início da lista ligada de antenas.

◆ removerAntena()

Antena * removerAntena ( Antena *  lista,
int  x,
int  y,
bool *  removido 
)

Remove uma antena localizada nas coordenadas (x, y).

Caso a antena exista, ela é removida da lista e a memória é libertada. O apontador booleano é atualizado para indicar sucesso ou fracasso.

Parâmetros
listaLista de antenas.
xCoordenada X da antena a remover.
yCoordenada Y da antena a remover.
removidoPonteiro booleano para status de remoção.
Retorna
Lista atualizada após a operação.
